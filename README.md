# Plonkin

**Short-term hiring, simplified.**

Powered by [Sau5](https://sau5.com)

## About

Plonkin connects businesses with skilled workers for short-term engagements (1-3 days). From restaurants needing extra hands to electricians tackling urgent jobs — hire fast, work flexible.

## Target Industries

- 🍽️ Restaurants & Hospitality
- ⚡ Electricians & Trades
- 🧹 Cleaning Services
- 💇 Hairdressers & Beauty
- 🚚 Delivery & Logistics
- 🎉 Events & Temp Staffing

## Global Focus

Available in multiple countries. Local talent, global platform.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
